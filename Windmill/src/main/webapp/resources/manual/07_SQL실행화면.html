7. SQL 실행화면

7.1 화면 구성

SQL 실행화면은 SQL 템플릿을 실행하고 결과를 확인하는 화면입니다.

SQL 실행 화면 스크린샷

7.2 기능 설명

1. 연결 선택

  • 드롭다운에서 데이터베이스 연결 선택
  • 사용자가 권한을 가진 연결만 표시
  • Shell 타입 템플릿의 경우 SFTP 연결 선택

2. 파라미터 입력

  • 템플릿에 정의된 파라미터 입력
  • 필수 파라미터는 * 표시
  • 파라미터 설명은 툴팁으로 표시

3. SQL 편집 (템플릿에 SQL이 없는 경우)

  • SQL 직접 입력
  • 문자 수 제한: 1,900,000자
  • 현재 입력 문자 수 표시

4. 실행

  • 실행 버튼 클릭
  • 로딩 표시
  • 실행 결과 표시

5. 자동 새로고침

  • 템플릿의 새로고침 타임아웃 설정에 따라 자동 실행
  • 일시정지 버튼으로 중지 가능
  • 일시정지 중에는 실행 버튼이 활성화됨

6. 결과 테이블

  • Tabulator 라이브러리 사용
  • 컬럼 정렬, 필터링 지원
  • 행 선택 가능
  • 스크롤 가능

7. 차트 표시

  • 결과가 차트 형식인 경우 자동으로 차트 표시
  • 차트 타입에 따라 다른 형태로 표시

8. 실행 정보

  • 실행 시간 표시
  • 결과 행 수 표시
  • 실행 상태 표시

9. 다운로드

  • 결과를 Excel, CSV 파일로 다운로드
  • IP 제한 적용 (환경설정에서 설정한 IP만 다운로드 가능)
  • 다운로드 불가 시 버튼 숨김김

10. 행 수 제한

  • 템플릿의 실행 제한 설정에 따라 결과 행 수 제한
  • 기본값: 1,000행
  • 최대값: 20,000행
