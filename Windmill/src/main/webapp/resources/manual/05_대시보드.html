5. 대시보드

5.1 화면 구성

대시보드 화면은 시스템의 전반적인 상태를 모니터링하고 시각화하는 화면입니다.

대시보드 화면 스크린샷

5.2 기능 설명

1. 연결 상태 모니터링

  • 각 DB 연결의 실시간 상태 표시
  • 정상: 초록색 배지
  • 오류: 빨간색 배지
  • 비활성: 회색 배지
  • 연결명, IP, 포트 정보 표시

2. 새로고침

  • 새로고침 버튼 클릭 시 연결 상태를 즉시 갱신
  • 자동 새로고침: 10초 간격으로 자동 갱신

3. 차트 자동 새로고침

  • 각 차트는 템플릿의 새로고침 타임아웃 설정에 따라 자동 갱신
  • 에러 발생 시 차트가 중단될 수 있음

4. 차트 에러 처리

  • SQL 실행 실패 시 에러 메시지 표시
  • 환경설정에서 에러 상태 리셋 가능
실행 기록 스크린샷

5. 메뉴 실행기록 조회

  • 최근 실행된 템플릿 목록
  • 실행 시간, 사용자 ID, 연결 ID, 템플릿명 표시
  • 실행 결과 요약 정보 표시
