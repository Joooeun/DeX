3. 사용자관리

3.1 화면 구성

사용자관리 화면은 시스템 사용자와 그룹을 관리하는 화면입니다. 관리자 권한이 있는 사용자만 접근할 수 있습니다.

사용자관리 화면 스크린샷

3.2 기능 설명

1. 검색 기능

  • 사용자 ID 또는 이름으로 검색
  • 검색 버튼 클릭 또는 Enter 키로 검색

2. 그룹 필터

  • 전체 그룹 또는 특정 그룹 선택
  • 선택한 그룹에 속한 사용자만 표시

3. 사용자 목록 테이블

  • 사용자 ID, 이름, 상태, 그룹, IP 제한, 마지막 로그인, 로그인 실패 횟수, 생성일 표시
  • 각 사용자별 수정/로그/삭제 버튼
  • 페이징: 한 페이지에 5명씩 표시

4. 사용자 생성/수정

새 사용자 버튼 또는 수정 버튼 클릭 시 사용자 생성/수정 모달이 표시됩니다.

5. 사용자 삭제

  • 사용자 목록에서 삭제 버튼 클릭
  • 확인 메시지 후 삭제 수행
  • 삭제된 사용자는 더 이상 로그인할 수 없습니다

6. 그룹 관리

그룹 추가 버튼 또는 수정 버튼 클릭 시 그룹 관리 모달이 표시됩니다.

3.3 모달

사용자 생성/수정 모달

사용자 생성/수정 모달 스크린샷

사용자 생성/수정 모달 필드

  1. 사용자 ID: 고유 식별자 (영문, 숫자, 언더스코어만 사용 가능)
  2. 이름: 사용자의 실제 이름
  3. 그룹: 사용자가 속할 그룹 (필수)
  4. 비밀번호: 로그인 비밀번호 (수정 시 비워두면 변경하지 않음)
  5. 상태: 활성/비활성/잠금
  6. IP 제한: 로그인 허용 IP 주소 (비워두면 모든 IP 허용, 예시: 192.168.1.* 또는 192.168.1.*,10.0.0.100)
  7. 저장 버튼: 사용자 정보를 저장합니다.
  8. 취소 버튼: 모달을 닫고 변경사항을 저장하지 않습니다.

비밀번호 설정 안내

  • 수정 시 비워두면 기존 비밀번호 유지
  • 비밀번호를 입력하면 임시 비밀번호로 설정되며, 사용자는 다음 로그인 시 비밀번호 변경이 강제됨

그룹 관리 모달

그룹 관리 모달 스크린샷

그룹 관리 모달 필드

  1. 탭 네비게이션: 그룹 정보 / 권한 관리 탭 전환
  2. 그룹명: 고유 이름 (필수)
  3. 설명: 그룹에 대한 설명
  4. 상태: 활성/비활성
  5. 메뉴 권한: 각 메뉴에 대한 접근 권한 설정 (체크박스)
  6. 연결 권한: 각 데이터베이스/SFTP 연결에 대한 접근 권한 설정 (체크박스)
  7. 저장 버튼: 그룹 정보를 저장합니다.
  8. 취소 버튼: 모달을 닫고 변경사항을 저장하지 않습니다.
