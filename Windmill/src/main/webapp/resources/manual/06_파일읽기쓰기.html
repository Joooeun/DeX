6. 파일읽기/쓰기

6.1 파일 읽기 화면

화면 구성

파일 읽기 화면 스크린샷

기능 설명

1. 파일 읽기

  1. SFTP 연결 선택
  2. 파일 경로 입력 (예: /home/user/file.txt)
  3. Submit 버튼 클릭
  4. 파일 내용이 에디터에 표시됨

2. 에디터 기능

  • Ace Editor 사용
  • 구문 강조 지원
  • 줄 번호 표시
  • 읽기 전용 모드

6.2 파일 쓰기 화면

화면 구성

파일 쓰기 화면 스크린샷

기능 설명

1. 파일 쓰기

  1. SFTP 연결 선택
  2. 파일 경로 입력 (예: /home/user/file.txt)
  3. 에디터에 파일 내용 입력/편집
  4. Submit 버튼 클릭
  5. 파일이 SFTP 서버에 업로드됨

2. 에디터 기능

  • Ace Editor 사용
  • 구문 강조 지원
  • 줄 번호 표시
  • 편집 가능 모드
  • 자동 완성 기능

3. 업로드 완료

  • 업로드 성공 시 알림 메시지 표시
  • 에디터 내용 자동 초기화
